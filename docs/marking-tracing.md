# Marking & Tracing

How to preview a job with the red-light **Trace** and run it with the laser.

!!! info "This page is being written"
    The outline below is what it will cover. The
    [your first mark](getting-started/your-first-mark.md) quickstart shows the basic flow.

## What this page will cover

- **Running a job** — **Run / Pause / Resume / Cancel**, and what happens at each stage.
- **Lens activation & homing gate** — the prompt that appears when the Z axis is enabled but not homed
  (home + run, or run anyway).
- **Trace (red-light preview)** — the modes:
    - **Full**, **Single Layer**, **Hull**, **Perimeter**, **Area** — what each previews and when to use it.
- **Progress** — the marking progress bar (group / layer / pass breadcrumb) and the time-remaining estimate.
- **3D slice marking** — slice order (top → bottom), per-slice sublayers, and **fill-through**.
- **Run isolation** — what's locked while a job runs (lens, axis-enable, destructive menu items) and why.
